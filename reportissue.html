Before reporting an issue, ensure it doesn't fall into one of the following categories:

  • Issues caused by other extensions interacting with my extensions.
  • Issues caused by using an off version of the scratch vm.
    (some extensions require specific mods of scratch to work).
  • Issues from misuse of the extension
    (refer to the documentation to ensure proper usage).
  • Issues with 3rd party apis
    (ex: Simple GPT3 going down)
  • Problems with debug blocks.
    These are for internal use only and may be removed at any time without notice.
  • Issues that have already been reported.
    (check the list here).

If your issue doesn't match any of these, you're ready to start making a report!
